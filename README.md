# face-verification-project

## Objectif
Comparer deux visages et déterminer s'il s'agit de la même personne.

## Méthodes
- FaceNet
- VGGFace2 (ResNet50)
- Similarité cosinus
- MTCNN (détection visage)

## Résultats
FaceNet > VGGFace2 en précision (AUC ≈ 1.0)
